# Learn Kana Quiz

🎯 **[Play the Live Demo here!](https://learn-kana-jp.netlify.app/)**

An interactive web application designed to help you practice and master Japanese Hiragana and Katakana.

_This project was heavily inspired by the excellent [Tofugu Kana Quiz](https://kana-quiz.tofugu.com/), but rebuilt from scratch with additional custom features and modern UI enhancements._

## 🚀 Features

- **Comprehensive Practice:** Test your knowledge of both Hiragana and Katakana alphabets.
- **Customizable Modes:** (Add your specific new features here, e.g., Timed mode, specific row selection, dark mode, etc.)
- **Instant Feedback:** Get immediate results on your Romaji inputs.
- **Clean Interface:** A modern, distraction-free design focused on learning.

## 🛠️ Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## 💻 How to Run Locally

Since this is a client-side web application, running it locally is extremely simple:

1. Clone this repository:
    ```bash
    git clone https://github.com/hazetage/learn-kana-jp.git
    ```
2. Navigate into the project directory:
    ```bash
    cd learn-kana-jp
    ```
3. Open the `index.html` file in your favorite web browser.
   _(Alternatively, you can use a local server like VS Code's "Live Server" extension for a better development experience)._
